# NeoNet Website Frontend

Frontend implementation of a responsive website for NeoNet, a local internet provider.

## Work Included

- Responsive HTML/CSS/JavaScript layout
- Home page, tariffs, promotions, support, and MEGOGO page
- Dark and light theme support
- Mobile navigation and adaptive footer
- Scroll-based section animations
- Tariff and promotion card layouts
- Mobile-first refinements for real device usage

## Tech Stack

- HTML
- CSS
- JavaScript

## Notes

This repository is intended to show the frontend implementation and structure of the project. It is not an official public distribution of the commercial website and is not intended for reuse, resale, or deployment by third parties without the website owner's permission.
